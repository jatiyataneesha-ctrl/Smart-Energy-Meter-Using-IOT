# Smart-Energy-Meter-Using-IOT
A real-time IoT-based Smart Energy Meter built using ESP32, capable of monitoring voltage, current, power consumption, and energy usage. The system sends live data to the cloud using the Blynk platform, enabling remote monitoring through a mobile app.

##  Features
-> Real-time voltage, current, power monitoring  
-> IoT-based remote access using Blynk  
-> Energy consumption tracking (kWh)  
-> Relay control for appliances  
-> LCD display support  

##  Blynk IoT Readings
| **Parameter** | **Value** |
| Voltage  | 220 V |
| Current  | 2.5 A |
| Power    | 550 W |
| Energy   | 0.75 kWh |
